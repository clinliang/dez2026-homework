
# Commit Convention

Format:
type(scope): message

## Types

- feat : new feature
- fix  : bug fix
- ref  : refactor
- exp  : experiment
- doc  : documentation
- chore: maintenance

## Examples

feat(ingest): add api loader
fix(db): handle null values
exp(llm): test prompt v2
doc(git): add branch strategy
ref(core): split pipeline logic
